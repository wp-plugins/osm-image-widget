== Description ==

This plugin lets you easily add images to widgets. Uses WordPress media library. You can add link, title, text, alternative text, CSS classes...

You can create amazing widgets by adding images. Also, you can add title, descriptive text and link to your widgets.

You can add CSS classes to customize the image and link button.

== Installation ==

Extract the zip file and just drop the contents in the <code>wp-content/plugins/</code> directory of your WordPress installation (or install it directly from your dashboard) and then activate it from Plugins page.
